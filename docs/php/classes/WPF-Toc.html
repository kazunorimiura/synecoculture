    TOC\MarkupFixer() 済みのコンテンツを返す。

    public get_the_content([string $more_link_text = null ][, bool $strip_teaser = false ]) : string

    get_the_content が the_content と同じ内容を返すようにするため、$this->markup_fixer 以外のロジックを the_content からコピペしている。これにより、テンプレート内でのコンテンツ の呼び出しを一度で済ませられるようにしつつ、get_html_menu を取得できるようにしている。

    Parameters

    Moreテキスト。

    $strip_teaser : bool = false

    本文の前にティーザー・コンテンツを掲載する。
